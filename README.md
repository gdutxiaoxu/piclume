# Piclume

Piclume is a browser-first image toolkit for compressing, converting, and resizing images online.

Website:
https://piclume.app

## Features

- [Compress images](https://piclume.app/compress-image)
- [Convert JPG, PNG, WebP, HEIC and AVIF](https://piclume.app/image-converter)
- [Resize images](https://piclume.app/resize-image)
- Browser-first processing
- Secure server fallback
- No account required
- Open product site: [https://piclume.app](https://piclume.app)

## Notes

- Core product source is not open sourced in this repository.
- This repository is a branding and communication space for Piclume.
